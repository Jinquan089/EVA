Tutorial de selectores de CSS

ID

Es un selector que se define por un identificador único al cual no debe repetirse en todo el documento. Su propósito es identificar el elemento al vincularlo en scripts u hoja de estilo

Tipo

Tipos de etiquetas son utilizados para dar un estilo o un formato a la pagina web que estamos creando

Selector de clase

El selector clase tienes como poner un ".clase" y luego pones los estilos que quieres poner.

Selectores Combinados

El selector combinados es que una etiqueta tenga más de un estilo

Selector combinados 2

los combinadores, que son un tipo especial de selectores en CSS que se usan en conjunto con selectores simples (clase, id, elemento) para hacer que una regla sea más específica.

Descendiente

Este selector afecta a todos los elementos que sean descendientes de otro elemento especificado, sin importar el nivel de profundidad al que se encuentren.

Hijo Directo

Este selector actúa sobre todos aquellos elementos que sean hijos de otro elemento especificado, pero que se encuentren en el primer nivel, es decir, si están dentro de otro elemento hijo de ese mismo padre, no serán tomados en cuenta

Elemento adyacente

Este selector afecta a los elementos que, teniendo el mismo elemento como padre, estén inmediatamente seguidos uno de otro, esta relación se representa con el símbolo + entre los selectores

Elemento Hermano

Este selector actúa sobre aquellos elementos que se encuentren precedidos por un elemento específico y que tengan como padre al mismo elemento, y se representa con el símbolo ~ entre los dos selectores

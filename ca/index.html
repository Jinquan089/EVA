Tutorial de selectors de CSS

ID

És un selector que es defineix per un identificador únic a el qual no ha de repetir-se en tot el document. El seu propòsit és identificar l'element a l'vincular-en scripts o full d'estil

Tipus

Tipus d'etiquetes són utilitzats per donar un estil o un format a la pàgina web que estem creant

Selector de classe

El selector classe tens com posar un ".clase" i després poses els estils que vols posar.

Selectors Combinats

El selector combinats és que una etiqueta tingui més d'un estil

Selector combinats 2

Els combinadors, que són un tipus especial de selectors en CSS que s'usen en conjunt amb selectors simples (classe, id, element) per fer que una regla sigui més específica.

Hi ha 4 tipus

Descendent

Fill directe

Adjacent

Germà

Descendent

Aquest selector afecta tots els elements que siguin descendents d'un altre element especificat, sense importar el nivell de profunditat a què es troben.

Fill Directe

Aquest selector actua sobre tots aquells elements que siguin fills d'un altre element especificat, però que es troben en el primer nivell, és a dir, si estan dins d'un altre element fill d'aquest mateix pare, no es tenen en compte

Element adjacent

Aquest selector afecta els elements que, tenint el mateix element com a pare, estiguin immediatament seguits un de l'altre, aquesta relació es representa amb el símbol + entre els selectors

Element Germà

Aquest selector actua sobre aquells elements que es trobin precedits per un element específic i que tinguin com a pare a el mateix element, i es representa amb el símbol ~ entre els dos selectors
